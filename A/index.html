Elementary +

HTML

Hypertext Markup Language, a standardized system for tagging text files to achieve font, colour, graphic, and hyperlink effects on World Wide Web pages.

Tag

A HTML Tag is also known as HTML Element it consists of an opening and closing brackets. Tags are made of triangular brackets. Closing brackets have a forward slash before the tag ID. Tags that do not require to be opened and closed are written like any opening set of brackets but with the forward slash at the end.

Browser

Short for Web browser, a browser is a software application used to locate, retrieve and display content on the World Wide Web, including Web pages, images, video and other files. As a client/server model, the browser is the client run on a computer that contacts the Web server and requests information. Examples of web browsers are Safari, Google Chrome, Firefox, Internet Explorer and Opera, some of which have mobile counter parts.

Web Application

A web application or web app is any application software that runs in a web browser and is created in a browser-supported programming language (such as the combination of JavaScript, HTML and CSS) and relies on a common web browser to render the application.

Web Server

Web servers are computers that deliver (serves up) Web pages. Every Web server has an IP address and possibly a domain name. For example, if you enter the URL http://www.pcwebopedia.com/index.html in your browser, this sends a request to the Web server whose domain name is pcwebopedia.com. The server then fetches the page named index.html and sends it to your browser.

Basic +

The World Wide Web Consortium (W3C) is the main decider in deciding which HTML Tags/Elements are 'in' or 'out'. Other deciders are web browser companies such as Apple, Google, Microsoft, Mozilla, etc... The largest contributors are the web browser companies because they write the tools that the average Joe usual to read the news and watch whatever videos he wants on the internet. This makes the major browser companies' job much easier when writing the frameworks for rendering the website on screen. Even though major browser companies decide on what HTML tags are used and how they appear, each browser has its own quirks such as different default margin sizes, which can make a web developer/designer's job more complicated at times.
