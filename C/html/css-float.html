CSS Float

Description

This video is very informative video on how floats work in CSS and its real life uses. It teaches us in detail how floats will place CSS elements on a websites. The video contains several demonstrations on how to use the float style to get the look that you want on your website. One of the demonstrations explain what happens when you use a right float rather than a left float, and how it moves the block elements on the page. Another demonstration show a real life situation where an unordered list is used to make a navigation bar. Overall, I think this video is a great introduction to using the float style in CSS.
