The IPO Model

The input-process-output (IPO) model, also known as the IPO+S model, is a functional model and conceptual schema of a general system. An IPO chart/table identifies a program’s inputs, its outputs, and the processing steps required to transform the inputs into the outputs.
IPO+S stands for Input Processing Output + Storage.
The IPO Model makes it possible for users to interact with these electronic devices in a standard way without much difference between systems. The IPO Model is essential as it is the most simple way that electronics can work with user input, and all electronic devices work with the IPO Model.
In the simple IPO model, information is 'inputted' and then sent to the processor which processes it/manipulates the data then sends the outcome of the processing to the output device. A simple example of this would be moving a mouse on a table. The mouse collects the motion data and sends it to the processor, which takes that data and moves the cursor then draws and image, which it then sends to a screen where you can see the outcome of your hand moving the mouse on the table.

CSS

CSS is short for Cascading Style Sheet and is used for placing and formatting a html website. It is used for describing the look and formatting of a document written in a mark up language.
CSS should be used over in-line formatting because it can be reused over many elements. CSS also has the ability to change the look a lot of elements without having to go through and change a setting in every element unlike in-line formatting.
It is advantageous to have an external style sheet in your website over having the CSS in side the html document because there will always be (an) element(s) that will share a common style. By importing an external style sheet and having only one style for the specific element (such as the header) instead of having its style information in each HTML document, makes it easier to make changes to the header throughout the whole website.

The IPSCO Model

The processing part of the IPO Model can be broken down into three sub sections for hardware - being Processing, Storage and Control. Thus the IPO model can become the IPSCO Model.
